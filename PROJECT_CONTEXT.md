Project: BarberIA

This project is a SaaS backend for barbershops.

Stack:
- Python
- Flask
- PostgreSQL
- SQLAlchemy

Features:
- Manage barbershops
- Manage barbers
- Manage clients
- Appointment booking
- Availability checking
- WhatsApp chatbot integration

Architecture:
- models/
- routes/
- services/

The system must prevent double booking and provide available time slots.
